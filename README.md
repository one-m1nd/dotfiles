# Dotfiles

> [!IMPORTANT]
> Setup `zsh` config first
> [my prezto fork](https://github.com/one-m1nd/prezto)

---

```bash
git clone git@github.com:one-m1nd/dotfiles.git "$HOME/.dotfiles"

cd "$HOME/.dotfiles"

./setup-dotfiles
```
